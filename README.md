# Система бронирования отелей на python

Простая система бронирования отелей с использованием ООП.

## Функционал
- Добавление номеров.
- Бронирование номера.
- Отмена бронирования.
- Автоматический расчёт стоимости.

## Информация

Этот проект был создан в обучающих целях для практики ООП
